
# CryptClip Ultimate v2

Supports:
- textarea
- input[type=text]
- input[type=password]
- React forms
- login forms
- framework-controlled inputs

Features:
- AES-GCM encryption
- Clipboard ciphertext only
- Real-time browser decryption

Install:
1. Extract ZIP
2. Open chrome://extensions
3. Enable Developer Mode
4. Load unpacked
5. Select folder

Test:
1. Save secret
2. Copy encrypted
3. Paste into login fields

Clipboard contains encrypted text only.
